Concepte de variable

Als llenguatges de programació fem servir variables per emmagatzemar informació. Sovint ens resulten molt útils per evitar haver de repetir i per fer càlculs.

Com declarar variables a SASS

A SASS definim variables com si fos una propietat de css però amb un $ al davant.

$variable: valor;

Recuperar valors

A SASS definim variables com si fos una propietat de css però amb un $ al davant.

SCSS CSS 
//variables
$base-color: #c6538c;
$border-dark: rgba($base-color, 0.88);

.alert {
  border: 1px solid $border-dark;
  //crida a la variable
}
 
.alert {
  border: 1px solid rgba(198, 83, 140, 0.88);
}
