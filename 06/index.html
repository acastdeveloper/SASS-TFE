Concepte de variable

Als llenguatges de programació fem servir variables per emmagatzemar informació. Sovint ens resulten molt útils per evitar haver de repetir i per fer càlculs.

Com declarar variables a SASS i com invocar-les

A SASS definim variables com si fos una propietat de css però amb un $ al davant.

$variable: valor;

Recuperar valors

A SASS definim variables com si fos una propietat de css però amb un $ al davant.

SCSS CSS 
//variables
$base-color: #c6538c;
$border-dark: rgba($base-color, 0.88);

.alert {
  border: 1px solid $border-dark;
  //crida a la variable
}
 
.alert {
  border: 1px solid rgba(198, 83, 140, 0.88);
}

Tipus de dades que admet SASS

  • Els mateixos que admet CSS.
    • Números i números amb unitats:
      $var: 1.2 | 13 | 10px | 1em ;
    • Cadenes amb i sense cometes:
      $var: "foo" | 'bar' | baz ;
    • Colors:
      $var: blue | #336699 | rgba(255, 0, 0, .5) ;
    • Llistes de valors separats per comes o espais:
      $var1: 1.5em 1em 0 2em | Helvetica, Arial, sans-serif ;
  • Altres tipus que trobem als llenguatges de programació:
    • Booleans: true | false:
      $var1: true | false ;
    • Null:
      $var1: null;
    • Mapes, d'un valor a un altre.:
      $var1: (key1: value1, key2: value2)

Diferències entre variables CSS i variables SASS

A un video d'EDTEAM Álvaro explica coses que es poden fer i les diferències entre les variables CSS i les variables SASS. Sense entretenir-me les poso a continuació:

  • Les variables css es poden fer servir dins de SASS(perquè a SASS pots fer servir CSS).
  • Les variables a CSS necessiten un scope(un àmbit). Si vols que aquella variable CSS que hem emmagatzemat sigui vista des de qualsevol element podem fer servir un pseudo-element anomenat :rootcom a selector que farà d'àmbit global. 
                :root {
              --color: green;
            }

    Posar :root és el mateix que utilitzar el selector html.

    No té per què ser un selector :root. Podria ser qualsevol altre. Per exemple:

                p {
              --color:green;
            }
  • L'àmbit, scope, de les variables és on existeixen.
    • Per a Sass aquest scope és de bloc de codi(és a dir que funciona entre claudators). Exemple: 
      :root { //La variable $color existeix des d'aquí...
  $color: green;

      body {
        background-color: $color;
      }

} // fins aquí. És a dir, en tot aquest bloc.
    • Per a CSS és a nivell de DOM (herència). 
      :root {
  --color: green;
}


body {
  background-color:var(--color);
}
  • A SASS les variables són estàtiques i a CSS són dinàmiques. Quan diem estàtiques vól dir que un cop s'han processat les variables SASS ja no canvien de valor. En canvi CSS permet canviar els valors de les variables i el navegador, en temps reial,
