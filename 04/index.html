1. Estalvi de selectors per anidació.

Per treballar si volem generar el codi de l'esquerra, que està plé de selectors repetits, podem anidar com al codi de la dreta.

estil01.css estil01.scss 
  main {
    background: white;
    max-width: 100vh;
  }
  main header h1 {
    font-family: 24px;
    font-weight: bold;
  }
  main header nav ul a {
    text-decoration: none;
    display: inline-block;
  }
  main header nav ul a:link {
    color: white;
    background: darkgrey;
  }
  main header nav ul a:visited {
    color: white;
    background: purple;
  }
  main header nav ul a:hover {
    color: white;
    background: darkorange;
  }
  main header nav ul a:active {
    color: white;
    background: red;
  }
  main section article {
    color: #333;
  }
  main section article pre {
    background: darkgreen;
    color: lightgrey;
  }
  main section article.pink {
    background: pink;
    color: darkgreen;
  }
  main section article.blue {
    background: blue;
    color: yellow;
  }
  main section article header {
    padding: 1em;
  }
  main section article header h2 {
    font-family: monospace;
  }

  main {
    background: white;
    max-width: 100vh;
    header {
      h1 { font-family:24px; font-weight: bold;}
      nav {
        ul {
          a {
            text-decoration:none;
            display: inline-block;
            &:link {color:white;background:darkgrey;}
            &:visited {color:white;background:purple;}
            &:hover {color:white;background:darkorange;}
            &:active {color:white;background:red;}
          }
        }
      }
    }
    section {
      article {
        color: #333;
        pre{ background:darkgreen; color:lightgrey;}
        &.pink {background:pink;color:darkgreen;}
        &.blue {background:blue;color:yellow;}
        header {
          padding: 1em;
          h2 {font-family:monospace;}
        }
      }
    }
  }

2. Selector "PARE" &.

Quan volguem referir-nos a l'element pare, haurem d'utilitzar el selector &.

  a {
    text-decoration: none;
    display: inline-block;
  }
  a:link {
    color: white;
    background: darkgrey;
  }
  a:visited {
    color: white;
    background: purple;
  }
  a:hover {
    color: white;
    background: darkorange;
  }
  a:active {
    color: white;
    background: red;
  }

  article {
    color: #333;
  }
  article pre {
    background: darkgreen;
    color: lightgrey;
  }
  article.pink {

  article.blue {
    background: blue;
    color: yellow;
  }
  article header {
    padding: 1em;
  }
  article header h2 {
    font-family: monospace;
  }
