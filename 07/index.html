Què són els mixins?

Els mixins són un tipus de funcions de SASS que agrupen conjunts de propietats i regles que agrupem en un bloc i predefinim per tal de ser reutilitzats cada cop que ens calgui. Van bé per estalviar-nos feina.

Com es declaren els mixins?

Mixins sense paràmetres

@mixin nom {
          propietats...
        }

Per definir un mixin que no li calguin paràmetres hem de col·locar @mixin seguit del nom que li vulguem posar al mixin per poder referir-nos a ell cada vegada. I després entre claudators posem les propietats que volem que tingui. A continuació veiem un exemple:

@include titulos;

Veiem un exemple a continuació:

SASS CSS 
@mixin titulos {
margin: 0 0 20px 0;
font-family: Georgia, serif;
font-size:20px;
font-weight:bold;
text-transform: uppercase;
}

section > h1 {
@include titulos;
color: #333;
}

section > h2 {
@include titulos;

}

section > h3 {
@include titulos;
color:#999;
}

Mixins amb paràmetres

@mixin nom($param1,$param2, $param3: DEFAULT) {
  propietats...
}

De la mateixa manera que indiquem a les funcions que li pasarem valors de paràmetres dins dels parèntesi. També podem establir valors per defecte al mixin.

I per invocar posen @include seguit del nom del mixin, i si hem de passar paràmetres hem de 

  @include titulos(18px, 600, #666);

Veiem un exemple complert:

SASS CSS 
@mixin titulos($fsize, $fw, $color: #000) {
  margin: 0 0 20px 0;
  font-family: Georgia, serif;
