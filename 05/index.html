1. Diverses línies: /* */

Aquest tipus de comentaris també el tenim a CSS. De fet, si fem un comentari d'aquest estil a l'arxiu .scss també s'hi veurà a l'arxiu .css

          /*
            Comentari
            escrit
            en diverses
            línies
          */
          /*
            Comentari
            escrit
            en diverses
            línies
          */

2. Una sola línia: //

Aquest tipus de comentaris serveixen per comentar el que hi ha a una sola línia, a partir d'escriure //, tal i com funciona a altres llenguatges de programació. De fet com que SASS és una mena de llenguatge de programació també l'accepta. Però no es veurà reflectit a l'arxiu css generat.

          body {
            font-family: sans-serif;
            // Comentari d'una línia.
          }
        
          body {
            font-family: sans-serif;
          }
