Estils ficticis

Es defineixen igual que qualsevol altre classe, però en comptes de fer servir un selector CSS normal li afegim el signe % al davant. Exemple:

  %estil_fictici {
  ... regles ...
  }

Nota:
Si NO "estenem" un estil fictici
llavors
NO es veurà aquest estil al full d'estils CSS que compili.

Estendre estils

A SASS podem heretar les propietats que ja tingui un altre estil. D'això en diem estendre. Per a estendre estils utilitzem la directriu @extend i indiquem de quina classe volem heretar les seves propietats. Nota: D'alguna manera ens recorda a l'extensió de classes de php. Fins i tot podem heretar les propietats d'una altra classe i aquelles que siguin diferents les podem sobre-escriure. Veiem algun exemple:


  %buttom {
  /*Aquest és un estil "fictici". Si no l'extenem no apareixerà al codi CSS de sortida. */
    padding:10px;
    color:#fff;
    background:blue;
    border-radius: 6px;
    display:inline-block;
  }

.cancelar {
  @extend %buttom;
  /* Directriu @extend per heretar les propietats d'una altra classe */

  background: red;
  /* Sobre-escriptura només de la propietat "background" */

}

.enviar {
  @extend %buttom;
  /* Herència */
  background: green;
  /* Sobre-escriptura*/
}
